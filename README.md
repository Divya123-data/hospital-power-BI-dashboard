# Hospital Operations Analytics Dashboard — Power BI

## Overview
Interactive 2-page Power BI dashboard tracking patient encounters 
and treatment costs for hospital operations reporting.

## What I Did
- Built KPI cards for Total Patients, Total Encounters, Total Treatment Cost
- Created 5 chart types — line chart, bar charts, donut chart, column chart
- Added 3 slicers (Gender, Date Range, Encounter Type) for dynamic filtering
- Built a drill-through page for disease-level patient and cost analysis

## Tools Used
Power BI — DAX, Power Query, Drill-through, Slicers, KPI Cards

## Key Insight
The dashboard lets a hospital operations manager identify which 
encounter types cost the most — useful for budget and resource planning
